# My Android App

This is a Movie Listing Android app using paging,hilt,kotlin,compose .

## Features
- Display list of movies with title , rating & poster from server

## How to Run
- Open the project in Android Studio.
- Sync Gradle.
- Run on an Android emulator or device.
